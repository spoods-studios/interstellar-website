|----------|-------------|
| Markdown body prose → `<meta>` attribute value | Git-tracked, developer-authored prose from four content trees is extracted and interpolated into HTML attribute context on 75 more pages |

No auth, no session, no user input, no database, no server. This plan adds no new mechanism —
it applies the mechanism Plan 03-01 built to the remaining routes.
