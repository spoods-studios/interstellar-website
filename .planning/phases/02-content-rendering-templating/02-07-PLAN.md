Write `src/lib/mdast-deepdive-links.mjs` exporting `createDeepDiveLinkPlugin({ resolvePhase })`, a Sätteri mdast plugin built with `defineMdastPlugin`, mirroring the wikilink plugin's shape. It must:
  - Do nothing unless the visitor context's file URL is inside the repo's roadmap tree. This plugin is registered globally on the Markdown pipeline, so scoping by path is what keeps it off the other three trees.
  - Visit `heading` nodes and, when the heading text matches a phase heading, record that phase number in the visitor context's per-document `data` bag. That bag is documented as shared across a single compile, which is what makes this safe without any plugin-level closure state.
  - Visit `text` nodes and, when the text matches the Discord-era deep-dive placeholder, replace it with a link node built from `resolvePhase(milestone, phaseNumber)` — the milestone coming from the file's name, the phase number from the recorded value. Give the link readable text naming the phase, not the placeholder wording.
  - Throw a message naming the file when a placeholder is found with no recorded phase number, or when `resolvePhase` returns nothing — the same loud-fail convention as D-10/D-39, never a silent passthrough of the Discord text.
  - `resolvePhase` is supplied from `astro.config.mjs` and walks `technical/<milestone>/` once to map phase numbers to deep-dive ids, returning the base-prefixed site URL. The base path arrives as an argument; no literal base string in this file.

Register the plugin in `astro.config.mjs` alongside the wikilink plugin in the same `satteri({ mdastPlugins: [...] })` array. Do not change the processor choice, the Shiki config, or the sitemap integration.

**Fallback if the stateful heading→text ordering proves unreliable at execution time:** render the roadmap body untouched and instead emit a generated per-phase deep-dive link block as chrome below the body, matching D-35's "generated, never hand-maintained" rule. Take this route only if the plugin cannot be made to work, and record it as an explicit deviation in the summary — the placeholder text must not survive onto the page either way.
